Sobre o Projeto

O NotaDez é um sistema completo de gerenciamento de notas acadêmicas desenvolvido para facilitar o controle de instituições, cursos, disciplinas, turmas e alunos.

Funcionalidades principais:

  • Gerenciamento de instituições e cursos
  • Cadastro de disciplinas e turmas
  • Controle de alunos e notas
  • Cálculo automático de médias com fórmulas personalizadas
  • Sistema de auditoria completo
  • Importação/Exportação de dados via CSV
